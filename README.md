# ubuntu-homelab
Documentation for an in-progress Ubuntu homelab where I'll store private files and future projects; documentation goes here to better keep track of my progress; as well as any details that I want to document for the future.
Actual SSH is on a private profile.

# Hardware
- Older Desktop Computer -- Pre-built PC (Merlin) connected to custom-built PC (Morgana)
- OS: Ubuntu 24.04.4 LTS x86_64
- Kernel: 6.8.0-111-generic
- Uptime: 17 hours, 12 mins
- Packages: 845 (dpkg)
- Shell: zsh 5.9
- Terminal: /dev/pts/0
- CPU: Intel i7-7700 (8) @ 4.200GHz
- GPU: NVIDIA GeForce GTX 1060 3GB
- Memory: 547MiB / 15935MiB

# Current Features
- Tailscale remote access
- File storage
- Media storage (movies, pictures)
  - Accessible from a remote server
- Accessed some installable programs for QoL ease of reading information and fun visuals.
  - btop: Displays all information of device not unlike a Task manager.
  - aafire / cmatrix: Animation softwares; uses text to create background visuals.
 
# Things I've learned so far
- SSH command line has access to remote machines with Tailscale's private network access.
- How to install files through this process.
